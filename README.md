# Бот-консул (Visa Assistant)

## О проекте
**Бот-консул** — чат-бот для граждан РФ, который помогает собрать актуальную информацию по визам и подготовке к поездке за рубеж. Он агрегирует **официальные требования** (консульства/визовые центры), **свежие отзывы** и **релевантные Telegram-апдейты**, а затем выдаёт **структурированный ответ**: пошаговый план, список документов, ссылки на анкеты/запись, типовые ошибки и советы по требованиям для въезда (страховка, билеты, проживание и т.д.).

## Зачем
Информация о визах разрозненная и быстро устаревает. Ручной поиск занимает много времени и часто приводит к ошибкам в документах и отказам. LLM + RAG позволяют быстро дать персонализированную инструкцию под конкретную страну и тип визы.

## Что умеет
- Подбирает требования под **страну / тип визы / город подачи / даты поездки**
- Формирует **чек-лист документов** и **пошаговую инструкцию**
- Даёт ссылки на **официальные сайты**, формы и запись
- Подсвечивает **подводные камни** и частые причины отказов
- Отмечает **актуальность** и различает «официально / по опыту»

## Данные и источники
- **Официальные сайты** консульств, визовых центров, миграционных служб  
- **Отзывы/кейсы** недавно получивших визу (форумы, публичные отчёты)  
- **Telegram-каналы** с оперативными изменениями и практическими советами  

Храним для каждого фрагмента: `источник`, `url/идентификатор`, `дата`, `страна`, `тип визы`, `текст`, `метки` (город/ВЦ/сборы/сроки и т.п.).

## Ключевые метрики
- Доля ответов, отмеченных как полезные
- Конверсия «запрос → полезный ответ»
- NPS
- Повторное использование
- Низкий процент фактических ошибок (контроль качества)

## Дисклеймер
Бот не является официальным источником. Правила могут меняться — в ответах всегда приоритет у официальных ссылок и указана дата актуальности.
